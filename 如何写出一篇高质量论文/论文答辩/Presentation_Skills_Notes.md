## 核心理念

> **It is NOT only a presentation.**
> **It is a performance!**

答辩不只是"汇报"，更是一场**表演**——内容和呈现方式同等重要。

## Part 1 — Outline of Presentation

### 时间分配参考（总计 15 分钟 + Q&A）

| 章节 | 建议时间 |
|------|:-------:|
| Introduction（引言） | 2 mins |
| Literature Review（文献综述） | 2 mins |
| Data & Methodology（数据与方法） | 2 mins |
| Research Results（研究结果） | **5 mins** |
| Discussion（讨论） | 3 mins |
| Conclusion（结论） | 1 min |
| **Total** | **15 mins** |
| **Q&A** | **10–15 mins** |

> 💡 Results 占用最多时间（5分钟），是答辩的核心展示部分。Q&A 时间与正式展示相当，需充分准备。

## Part 2 — General Structure

### 答辩各章节内容要点

| 章节 | 需要展示的内容 |
|------|--------------|
| **1. Introduction** | • 给出研究主题的**简要背景**（brief background）<br>• 明确你的**研究问题**（research questions）<br>• **激发观众兴趣**（motivate），让他们愿意专注听你的汇报 |
| **2. Literature Review** | • 列出你论文中引用的**最相关文献**，标注作者和年份，简述其核心发现<br>• 如有共识，给出该领域的**主流观点总结**（majority view） |
| **3. Research Method** | • 解释你用来回答研究问题的**研究方法** |
| **4. Data** | • 解释你的**数据来源**（sources）<br>• 解释你如何**收集和分析**数据 |
| **5. Research Results** | • **报告并评估/分析**你的研究结果 |
| **6. Discussion** | • 讨论研究的**局限性**（limitations）<br>• 提出**未来研究方向**（areas for further research） |
| **7. Conclusion** | • 总结你的**主要发现**（main findings）及其**意义和影响**（implications） |

## Part 3 — Some Tips

### 🖥️ 幻灯片设计技巧

**结构逻辑：**
- 开头给出简要 **overview**（概述），然后呈现内容，最后 **review**（回顾）重点
- 每张幻灯片只传达 **one idea**（一个核心观点）

**文字规范：**
- **Simplify and limit** the number of words on each screen（精简文字，控制数量）
- 使用关键短语，只包含 **essential information**（必要信息）
- 使用 **bullet points**，但不要过多
- **不要**把大段文字搬上幻灯片——幻灯片是提示卡，不是讲稿！

**视觉设计：**
- 使用**对比色（contrast color）**确保文字和背景清晰可辨
- 使用**必要的图表（essential tables）**，复杂数据用图表传达更高效
- **避免过度动画**（animate everything 是大忌）

**数量控制：**
- **Limit the number of slides**（控制幻灯片数量）——频繁翻页会让观众失去耐心
- 经验法则：***one slide per minute***（每分钟一张幻灯片）

---

### ❌ Bad Slide 特征（避免踩坑）

| 错误类型 | 说明 |
|---------|------|
| 文字过多 | 一张幻灯片塞满段落文字，观众忙着读而不是听你说 |
| 字体过小 / 颜色不对比 | 投影后无法清晰辨认 |
| 全文大写 | ALL CAPS 比正常大小写更难阅读 |
| 过度强调 | 大量下划线、粗体、不同字体混用，反而杂乱 |
| 拼写/语法错误 | 直接损害专业形象 |
| 幻灯片数量过多 | 如同 "Slide #1/97"——观众早已入睡 |

---

### 🗣️ 现场表达技巧

| 技巧 | 说明 |
|------|------|
| **聚焦主线** | Restrict to **main arguments and methods**，不要迷失在细节中 |
| **眼神交流** | Keep your **eyes on the audiences**，不要只盯着屏幕或稿子 |
| **控制语速** | Do not talk too fast；适时停顿；speak **comfortably and clearly** |
| **展现自信** | Present with **confidence** |
| **承认局限** | *Admit* you do not have all the answers——坦诚局限反而能提升**可信度（credibility）** |
| **专业着装** | Dress **professionally** to respect your audiences（商务正装） |

### ⭐ 最重要的一条

> ***Practice, practice, practice!***
>
> 重要的事情说三遍——**反复练习**是成功答辩的最关键因素。

## 全课总结

```
答辩 = 内容（Content）+ 呈现（Delivery）+ 练习（Practice）

内容准备：            幻灯片设计：          现场表现：
7段标准结构          简洁 / 对比色          眼神接触
时间合理分配         1张≈1分钟             语速适中
重点在 Results       一张一观点             承认局限
Q&A充分准备          禁大段文字             专业着装
```

| 维度 | 核心要点 |
|------|---------|
| **时间分配** | 总15分钟；Results最重要占5分钟；Q&A 10-15分钟 |
| **内容结构** | 引言→文献→方法→数据→结果→讨论→结论，逐层推进 |
| **幻灯片** | 简洁、对比色、每页一观点、1分钟/张、禁止文字堆砌 |
| **表达** | 自信、眼神接触、语速适中、坦承局限、专业着装 |
| **最重要** | ⭐ **Practice, practice, practice!** |
